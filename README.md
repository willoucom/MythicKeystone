# MythicKeystone LibDataBroker

**Your mythic keystones, right in your Titan Panel or Fubar bar.**

MythicKeystone_LibDataBroker exposes your current keystone as a broker data source and adds a tooltip showing all your alts' and guild members' keystones, grouped by dungeon.

---

## Features

- **Broker text** — displays your current key level and dungeon name in any LibDataBroker display (Titan Panel, Fubar, Bazooka, etc.)
- **Click** — opens / closes the LFG frame
- **Tooltip** — shows alts and guild keystones grouped by dungeon, with weekly run stats per character
- **Addon Compartment** — also registers in the WoW addon compartment menu with the same tooltip

---

## Requirements

- **[LibMythicKeystone](https://legacy.curseforge.com/wow/addons/libmythickeystone)** — required
- **[MythicKeystone](https://legacy.curseforge.com/wow/addons/mythickeystone)** — required
- A LibDataBroker display addon (Titan Panel, Fubar, Bazooka…)
